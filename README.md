# Football Analytics Portfolio

A collection of football analytics projects built using the StatsBomb Open Data and inspired by the Soccermatics course.

## Goals

- Learn football analytics from the ground up
- Build reusable analysis tools
- Reproduce and improve Soccermatics examples
- Build portfolio-quality visualizations

---
